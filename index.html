🎯 Format interclub

Le tournoi interclub oppose des équipes de 2 joueurs fixes. Les partenaires ne changent pas de toute la journée.

  • Chaque équipe est composée de 2 joueurs représentant un club.
  • Les 2 joueurs restent ensemble pour tous les matchs.
  • En configuration 2 salles, chaque équipe reste dans la même salle toute la journée.
📅 Programme
  • Le tournoi se déroule en deux sessions : matin et après-midi.
  • Le nombre de tours est configurable par l'organisateur (ex. 3 matin + 4 après-midi).
  • Une coupure déjeuner peut être prévue entre les deux sessions.
  • Les terrains sont remplis au maximum à chaque tour.
⚖️ Appariement
  • Même club évité : deux équipes du même club ne s'affrontent pas si possible.
  • Répétitions minimisées : on évite qu'une même paire se rencontre deux fois.
  • Équilibre de niveau (si activé) : équipes de force similaire opposées.
  • Si le nombre d'équipes est impair dans une salle, une équipe est exemptée à rotation.
🏆 Points et classement
  • Victoire = 3 pts
  • Match nul = 2 pts chaque équipe
  • Défaite = 0 pt
  • Exemption (repos) = 2 pts de compensation

Départage à égalité de points

  1. Goal-average général
  2. Confrontation directe
  3. Buts marqués
  4. Nombre de victoires
  5. Matchs joués
